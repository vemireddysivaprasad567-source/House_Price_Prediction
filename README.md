# House Price Prediction

## Project Overview

This project aims to predict house prices based on features like area, number of bedrooms, and location using Machine Learning algorithms.

## Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
##  Dataset
The dataset contains the following features:
* Area (in square feet)
* Number of bedrooms
* Location
* Price

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Model Building (Linear Regression)
5. Model Evaluation
6. Prediction

## Machine Learning Model
* Linear Regression

## Future Improvements
* Add more features (location, amenities)
* Use advanced models (Random Forest, XGBoost)
* Deploy using Streamlit
